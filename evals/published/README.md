# Published evaluation evidence

No complete scored evaluation package is published yet.

The repository contains prepared external runs for cases `03-batch-file-converter-cli` and `04-high-throughput-event-ingestion`, but those runs are only `EVALUATION_PREPARED`; they do not contain recorded `evaluation.json` files or scores. Publishing candidates as scored evidence would fabricate evidence, so this directory intentionally stops at an honest status statement.

When a real completed run exists, publish only unchanged candidates, neutral candidate IDs, verified input/snapshot/rubric hashes, evaluator artefacts, comparison output, and provenance. The package must state that nondeterministic future model runs may differ byte-for-byte.
