# Failure taxonomy

Allowed tags:

`OVERENGINEERING`, `UNDERENGINEERING`, `TOOL_FIRST_SELECTION`, `FALSE_ENFORCEMENT_CLAIM`, `MISSING_CORE_INVARIANT`, `UNSUPPORTED_CERTAINTY`, `FAILURE_SEMANTICS_GAP`, `SECURITY_BOUNDARY_GAP`, `DATA_OWNERSHIP_GAP`, `MIGRATION_GAP`, `OPERATIONAL_MISMATCH`, `MODE_MISCLASSIFICATION`, `REPOSITORY_EVIDENCE_GAP`, `DECORATIVE_DOCUMENTATION`, `UNJUSTIFIED_JAVA_SELECTION`, `WEB_STACK_LEAKAGE`.

Each finding includes severity, confidence, summary, and evidence. Use the smallest number of tags that explain the failure.
